## Installation

**Don't `npm install` this directly**

```bash
$ npm init egg --type empty showcase
$ cd showcase
$ npm install
$ # then run with `npm run dev` / `npm start` / `npm test`
```

See [egg-init](https://github.com/eggjs/egg-init) for more detail.
